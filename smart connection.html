week 1

In de eerste week van Smart Connection kregen we veel introductie van het vak. Later is de les onderverdeeld in groepjes van 5 personen. Samen met mijn groepje heb ik een verband gelegd tussen de termen van Smart Connection. Het bord is hieronder weergegeven:


Aan het eind van deze opdracht presenteerde de groepjes hun gemaakte blad. Omdat ik nog veel vragen had heb ik besloten om te presenteren en daarna feedback te ontvangen. Door te presenteren heb ik meer informatie opgedaan over Smart Connection.

week 2

Tijdens les twee van Smart Connection was ik vooral bezig met het ontdekken van HTML en CSS. Voorafgaand aan de les heb ik twee video's bekeken, een over HTML en een over CSS. Door me voor te bereiden op de les kon ik deze makkelijker volgen. Tijdens deze les heb ik vooral vragen gesteld over de mogelijke toepassingen van HTML.

week 3

Tijdens de derde les zijn de functionaliteiten van JavaScript besproken. Tijdens deze les vond ik het moeilijk een verband te leggen tussen JavaScript en html. Om dit duidelijk te maken heb ik vragen aan Ryan gesteld over de toepassingen van JavaScript.

Tijdens het tweede deel van de les werdt de opdracht van het bedrijf ASML gepitcht. ASML is een bedrijf wat machines maakt om Chips te produceren. Samsung is een grote klant van ASML. ASML had de opdracht om communicatie tussen teams over de gehele wereld te verbeteren. Cultuurverschil, tijd en het gebrek aan contact beïnvloed de communicatie nadelig.

week 4

Tijdens deze les is er uitleg gegeven over het informatie doorsturen en opslaan in een database. Met behulp van webdev3 is dit klassikaal gemaakt Dit is vooral de basis. na deze les ben ik zelf een contactformulier op gaan stellen

Het contactformulier op deze website is te vinden op de home page. Je kan je subscribe op mijn website. Om dit te maken heb ik het programma Mail Chimp gebruikt. Mail Chimp is een server die je kan geben om bezoekers gegevens op te slaan. Dit vond ik een handig programma, het geeft de mogelijkheid om al je contacten in een keer een mail te sturen. In deze mail kan je een vragenlijst invullen. De antwoorden worden door Mail Chimp verzameld in een overzichtelijke grafiek

week 6

Door het corona virus is de les in week 6 komen te vervallen. In plaats van de reguliere les heeft Witek een masterclass over machine learning gegeven.

Machine Learning bestaat uit 3 stappen.
gegevens verzamelen en plaatsen in grafieken en rapporten.
patronen vinden en voorspellen.
mogelijkheid tot gebruiken informatie.

Business context Veel bedrijven zijn bezig met Big Data. De meeste bedrijven willen zo veel mogelijk data verzamelen. Het verzamelen van de data wordt al veel gedaan, het porbleem is dat bedrijven niet weten hoe ze deze data kunnen inzetten. Machine Learning is hier een goede oplossing voor. Met Machine Learning worden gegevens al geordend. Overeenkomsten en verschillen kunnen gemakkelijk worden gezien. Bedrijven kunnen met Machine Learning strategisch omgaan met de verkergen data. Machine Learning is waardevol voor elk bedrijf.

Reflectie Smart Connection

Omdat ik totaal geen ervaring heb met ICT had ik geen voorkennis van Smart Connection. De eerste weken waren voor mij behoorlijk zwaar in het begin was alles erg onduidelijk. Om deze reden heb ik actief gestudeerd en me goed voorbereid op de lessen door video's te kijken. In de eerste weken was ik vooral bezig om HTML en CSS te begrijpen. De weekopdrachten waren voor mij nog best moeilijk. Door uiteindelijk de weekopdrachten goed te maken ben ik HTML en CSS beter gaan begrijpen. Tijdes de laatste lessen ben ik vooral mijn portfolio gaan verbeteren en HTML gaan combineren met CSS

Na de lessen ben ik ook verder gegaan met het verbeteren van mijn portfolio. Doordat het programmeren nieuw voor me was vond ik het erg interessant. km
